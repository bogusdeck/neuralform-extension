EXTENSION FEATURES
==================

✅ Auto-fills 14+ field types
✅ Smart field detection (name, id, label, placeholder, aria-label)
✅ Beautiful gradient UI with 3 tabs
✅ Local browser storage (privacy-first)
✅ Visual feedback (green highlights)
✅ One-click form filling
✅ Works on any website
✅ Clear data option
✅ Zero external dependencies

QUICK START
===========

1. Run setup script:
   $ ./setup.sh

2. Choose option 1 (quick test)

3. Load in Firefox:
   about:debugging → Load Temporary Add-on → select manifest.json

4. Test on:
   - test-form.html (included)
   - Google Forms
   - Any contact form
